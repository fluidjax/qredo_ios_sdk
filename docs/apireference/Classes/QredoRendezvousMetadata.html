QredoRendezvousMetadata

@interface QredoRendezvousMetadata : NSObject

Contains information about a previously created Rendezvous. Used to retrieve the QredoRendezvousRef

QredoRendezvousMetadata objects are returned by enumerateRendezvousWithBlock:completionHandler

Other Rendezvous information can be accessed from the QredoRendezvous
